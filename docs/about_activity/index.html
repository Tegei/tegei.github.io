Blender Activity

主に初学者を中心にBlenderの技術を学びます。
Blenderは無料で使用できる3DCGソフトです。

活動は週に１回で、その回ごとに設定されたテーマに沿って勉強します。

・オブジェクト作成
・シュミレーション
・画像の作成/編集
・動画の作成/編集
など

慣れてきたら、各々でモデル、画像、動画など様々な作品作りに取り掛かります.
テーマは幅広く扱うので、最終的にはBlenderのほとんどの技術に触れることになります。

他ではなかなか扱わないので、少しでも気になる方は参加してみてください！

Unity Activity

Unityで各々ゲームを作ります。
活動は週に１回で、チュートリアルのゲームを制作しながらUnityの技術を勉強します。
チュートリアルゲームの制作や改造をしていくうちに基本的な操作を身に着ける事ができます。
また、途中経過や完成品をメンバーに共有する機会もあります。

ちなみにメンバーの中にはUnityを使ってゲーム製作のイベントに参加している人もいます。

個人でのゲーム製作に興味ある方にはオススメです！

Design Activity

Web Activity

手芸のホームページ更新をします。
序盤は基礎的なWebプログラミングの勉強会を週に１回行います。
学んだことや調べたことを生かし、最終的には手芸のホームページを班員で一新します。

Webを初歩から学びたい人からより力をつけたい人まで、様々な人が活動しています！
Webの知識や技術に触れたい人、大募集中です！

Movie Activity

動画を作成し、サークル内に共有します。
一定期間ごとにテーマを設けて、そのテーマに沿った動画を各々で作成・共有しています。
新入生歓迎用のPVや学祭で披露される動画は基本的にここのメンバーが作成しています。

それぞれの作品や、動画編集のコツなどを他の人と気軽に共有できます。
動画制作に少しでも興味があれば、初心者・経験者問わず大歓迎です！
