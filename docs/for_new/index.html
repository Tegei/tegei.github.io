入部の流れ

  1. Twitterや以下のフォームから連絡
  2. 中の人が確認し、学籍番号宛にメールを送信
  3. メールに付随するリンクからDiscordサーバーに！

入部/質問連絡フォーム

入部や質問、その他連絡がある場合はこちらのフォームよりお願いいたします。
質問のみの場合でも、回答の為学籍番号をご入力ください。
(学内の方でない場合は、メールアドレスをお願いします。)
